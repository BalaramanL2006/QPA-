2. KEY FUNCTIONS IN app.py
================================================================================

load_history()
├─ Purpose: Load history.json into memory on app startup
├─ Location: Called in if __name__ == '__main__'
├─ Returns: Populates global history_data variable
└─ Handles: Missing files, corrupted JSON, file errors

save_to_json(filename, level, easy, medium, hard, easy_count, medium_count, hard_count, total_questions)
├─ Purpose: Save analysis result to history.json
├─ Location: Called from save_analysis() after analyze()
├─ Updates: Both history.json file and global history_data
├─ Handles: File creation, existing data, concurrent writes
└─ Returns: Prints success/error message

@app.route('/history')
├─ Purpose: Render history page
├─ Method: GET
├─ Returns: history.html with history_data variable
└─ Sample: http://localhost:5000/history

@app.route('/delete_history/<int:index>', methods=['GET', 'POST'])
├─ Purpose: Delete specific history entry by index
├─ Method: GET/POST
├─ Parameter: index (zero-based position in list)
├─ Action: Removes entry and updates history.json
└─ Redirects: Back to /history

@app.route('/clear_all_history', methods=['GET', 'POST'])
├─ Purpose: Delete all history entries
├─ Method: GET/POST
├─ Action: Clears history_data and resets history.json to []
└─ Redirects: Back to /history

3. JINJA2 TEMPLATE STRUCTURE IN history.html
================================================================================

CONDITIONAL DISPLAY:
{% if history and history|length > 0 %}
  ├─ Checks if history list exists and has items
  ├─ If TRUE: Shows history grid with all entries
  └─ If FALSE: Shows "No Analysis History" empty state

LOOP THROUGH HISTORY:
{% for item in history %}
  ├─ Iterates through each analysis in history list
  ├─ Provides loop.index0 for delete buttons (zero-based index)
  └─ Provides access to all item properties

DISPLAY EACH ENTRY:
{{ item.filename }} → Paper name
{{ item.date }} → Analysis timestamp
{{ item.level }} → Difficulty level
{{ item.easy }}, {{ item.medium }}, {{ item.hard }} → Percentages
{{ item.easy_count }}, {{ item.medium_count }}, {{ item.hard_count }} → Counts
{{ item.total_questions }} → Total questions

CONDITIONAL STYLING:
{% if item.level == 'Easy Paper' %}
  ├─ Shows green emerald-500 badge
{% elif item.level == 'Hard Paper' %}
  ├─ Shows red-500 badge
{% else %}
  └─ Shows blue-500 badge for Moderate Paper

DELETE BUTTONS:
href="/delete_history/{{ loop.index0 }}" → Individual delete
href="/clear_all_history" → Delete all

6. HOW TO USE
================================================================================

1. ANALYZE A PAPER:
   - Go to http://localhost:5000
   - Upload a paper or paste text
   - Click "Analyze"
   - View results with difficulty breakdown

2. VIEW HISTORY:
   - Click "History" in navbar
   - See all past analyses
   - Each card shows: filename, date, difficulty level
   - Click "View Breakdown" to expand details

3. DELETE A SPECIFIC ENTRY:
   - Go to History page
   - Click trash icon on any card
   - Confirm deletion
   - Entry removed from history.json and display

4. DELETE ALL HISTORY:
   - Go to History page
   - Click "Clear All History" button at top
   - Confirm deletion
   - All entries removed, shows empty state
