                         🚀 HOW TO GET STARTED 🚀
═══════════════════════════════════════════════════════════════════════════════

WINDOWS USERS (Easiest):
  1. Open file manager
  2. Navigate to: question-paper-analyzer folder
  3. Double-click: run.bat
  4. Wait for setup (first time only: ~30 seconds)
  5. Browser opens automatically → Start analyzing!

MAC/LINUX USERS (Easy):
  1. Open Terminal
  2. cd question-paper-analyzer
  3. chmod +x run.sh && ./run.sh
  4. Wait for setup (first time only)
  5. Browser opens automatically → Start analyzing!

MANUAL START (All Systems):
  1. Open Terminal/Command Prompt
  2. cd question-paper-analyzer
  3. python -m venv venv
  4. Activate:
     • Windows: venv\Scripts\activate
     • Mac/Linux: source venv/bin/activate
  5. pip install -r requirements.txt
  6. python app.py
  7. Open browser: http://127.0.0.1:5000

→ Delete database.db and restart (it recreates)
